# Assets 文件清单

请将以下文件放入此文件夹，文件名需要完全一致：

| 文件名 | 用途 | 格式建议 |
|--------|------|---------|
| `tomato-app.png` | 番茄钟 APP 截图 | PNG，手机截屏即可 |
| `voInko-video.mp4` | VoInko 产品推广视频 | MP4，建议45秒以内 |
| `voInko-poster.png` | 视频封面图（可选） | PNG，16:9 比例 |
| `qr-voInko.png` | VoInko 公众号二维码 | PNG，正方形 |
| `qr-personal.png` | 个人微信公众号二维码 | PNG，正方形 |

## 使用说明

1. 将文件放入此 `assets/` 文件夹
2. 用浏览器打开 `index.html`
3. 按 `←` `→` 键翻页，按 `F` 全屏

## 键盘快捷键

- `←` / `↑` / `PageUp`：上一页
- `→` / `↓` / `Space` / `PageDown`：下一页
- `Home`：回到第一页
- `End`：跳到最后一页
- `F`：全屏切换

## 手机/平板

支持左右滑动翻页。
